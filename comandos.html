Comandos Básicos no Ubuntu

  • sudo apt update: Atualiza a lista de pacotes disponíveis nos repositórios.
  • sudo apt upgrade: Atualiza todos os pacotes instalados para a versão mais recente.
  • ls: Lista os arquivos e diretórios no diretório atual.
  • cd: Muda o diretório de trabalho atual. Exemplo: cd /var/www
  • mkdir: Cria um novo diretório. Exemplo: mkdir novo_diretorio
  • rm: Remove arquivos ou diretórios. Exemplo: rm arquivo.txt
  • chmod: Modifica as permissões de arquivos. Exemplo: chmod 755 arquivo.sh
