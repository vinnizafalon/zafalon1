O que são páginas estáticas?

Uma página estática é uma página da web cujo conteúdo é fixo e não muda automaticamente, ou seja, é criada uma vez e permanece igual até que alguém faça uma alteração manual. Elas são normalmente escritas em HTML, CSS e, em alguns casos, JavaScript.

Esse tipo de publicação é ideal para sites que não necessitam de interatividade em tempo real ou conteúdo dinâmico, como blogs simples ou páginas institucionais.
